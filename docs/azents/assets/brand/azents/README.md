# Azents Brand Asset Drafts

Draft logo assets generated from the May 31, 2026 Azents logo exploration.

These are not wired into the product yet. They are stored here as high-resolution
source candidates for later product, app icon, landing, and social usage.

## Files

- `azents-icon-transparent.png` — 4096x4096 standalone `Az` mark on transparent background.
- `azents-icon-dark-tile.png` — 4096x4096 optional dark rounded-square app icon treatment.
- `azents-logo-horizontal-transparent.png` — 4096x861 horizontal `Azents` logo on transparent background.
- `azents-logo-banner-light.png` — 3840x1200 horizontal logo on a light banner canvas.
- `azents-brand-sheet-source.png` — original generated exploration sheet.
- `azents-brand-sheet@2x.png` — upscaled exploration sheet for visual reference.
- `azents-assets-preview.png` — quick contact sheet for reviewing the exported asset set.

## Direction

- Exact wordmark reading: `Azents`.
- Composition: `Az` ligature mark functions as the first two letters, followed by `ents`.
- Color intent: graphite mark with a quiet cool-white light reflection; the gradient is a light effect, not a multi-color brand fill.
- Default icon usage should prefer transparent background. The dark tile is optional.
