# Never Hand-Edit `src/generated/`

The generators rewrite the entire `src/generated/` tree on every run. Hand-edits silently disappear and create the illusion of a working contract that's actually inconsistent with the upstream spec.

- AVOID editing files under `packages/azents-admin-client/src/generated/` or `packages/azents-public-client/src/generated/`.
- If you need to change generated code, change the OpenAPI spec upstream and rerun `turbo run generate --filter=<package>`.
- For client-side wrappers/customizations, add a hand-written file next to `generated/` (e.g. `src/api-client.ts`) that imports from generated.

## Workflow

```bash
# 1. Backend changes routes → openapi.json is regenerated by the azents pre-commit hook.
# 2. Regenerate clients.
turbo run generate --filter=@azents/public-client
turbo run generate --filter=@azents/admin-client
# 3. Commit the regenerated files alongside your changes.
```
