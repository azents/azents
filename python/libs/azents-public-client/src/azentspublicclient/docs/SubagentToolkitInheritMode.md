# SubagentToolkitInheritMode

Subagent toolkit inherit mode.  Stored at Agent row (agents table) level (DP1 A). Consistent with Model inherit at the same level.  - ``NONE``: Use only subagent own ``agent_toolkits`` (existing behavior). - ``ALL``: Use only parent toolkit and ignore subagent own toolkit.   (DP6 exclusive).

## Enum

* `NONE` (value: `'none'`)

* `ALL` (value: `'all'`)
