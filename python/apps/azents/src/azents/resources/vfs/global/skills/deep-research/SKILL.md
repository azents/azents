# Deep Research

Use this Skill when the request requires current, niche, contested, or multi-source evidence.

1. Frame the question, scope, and decision criteria before searching.
2. Prefer primary sources for technical and factual claims.
3. Compare dates, definitions, methods, and incentives across sources.
4. Separate observed facts from inference and unresolved uncertainty.
5. Cite the evidence next to the claims it supports.
6. End with a direct synthesis that answers the user's actual question.
